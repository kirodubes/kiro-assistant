# Kiro Video Tutorials

Erik Dubois publishes Kiro tutorials on YouTube. Use this file to point
users at the right video. Suggested order when helping someone:

1. **Start with the website videos** below — the curated starting point
   hand-picked for kiroproject.be, the beginning of the Linux/Kiro
   journey.
2. Then the **KIRO** series — the current, numbered Kiro tutorial series.
3. Then the older **Kiro** videos for release history and earlier how-tos.

Channel: https://www.youtube.com/@ErikDubois

> When you can't find an exact match here, **send the user to search the
> channel** — see *Search the channel* at the bottom. That is the single
> most useful thing you can do; the catalog is far larger than this list.

## Older videos still apply more often than you'd think

The channel has thousands of older videos (including the ArcoLinux era)
that are not listed here. Don't dismiss them: Kiro runs the same apps,
with the same or only slightly different configuration, so a great deal
still applies directly. Many are also **workflow** videos and
**common-sense, analytical** videos — the thinking carries over to Kiro
unchanged. If a topic isn't covered above, an older video may answer it.
