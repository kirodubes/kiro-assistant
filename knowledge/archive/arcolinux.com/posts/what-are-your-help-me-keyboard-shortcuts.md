# What are your help me keyboard shortcuts

![](https://arcolinux.com/wp-content/uploads/2018/01/archmerge-keyboards.jpg)

There are many shortcuts build into

  * Xfce
  * Openbox
  * i3



Many of them are the same. So once you know them you can use them everywhere.

I want you to know these 3 already now.

> They are the "get me out of here" shortcuts.

Keyboard Shortcuts To Remember

SUPER + Escape

Kill the application now

Keyboard Shortcuts To Remember

CTRL + ALT + BACKSPACE

Kill xserver - back to login screen

Keyboard Shortcuts To Remember

CTRL + shift + escape

taskmanager

**Where do I find the other keyboard shortcuts via the menu?**

  * Xfce menu : **Keyboard**
  * Openbox menu : via the menu **Display Keybinds** or via the menu **Preferences/Openbox/Gui Keybinds**
  * i3 : config in ~/.config/i3 - no gui available in i3



**Where do I find the other keyboard shortcuts via the files they are stored?**

  * Xfce menu : **~/.config/xfce4/xfconf/xfce-perchannel-xml/xfce4-keyboard-shortcuts.xml**
  * Openbox menu : **~/.config/openbox/rc.xml**
  * i3 : **~/.config/i3**
