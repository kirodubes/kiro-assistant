# How to install minecraft on ArcoLinux

![](https://arcolinux.com/wp-content/uploads/2020/03/minecraft.png)

You can install minecraft from the ArcoLinux repositories.

sudo pacman -S minecraft-launcher

We show you in the video

  * how to install it
  * choose the java version
  * login
