# Installing feedthebeast modpacks for minecraft on any Arch Linux system

![](https://arcolinux.com/wp-content/uploads/2019/07/feedthebeast.jpg)

Work in progress

I had the feeling feed the beast did not need any minecraft-server or spigot server so I uninstalled both of them and started minecraft via the menu with feedthebeast.

Feedthebeast will install its own server to play on.
