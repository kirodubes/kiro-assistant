# Installing software with pamac or gnome-software

We can use either pamac or gnome-software to install packages on ArcoLinux. We are using applications that give us a graphical representation of ALL software that is available.

Once you know the name, you can use the terminal to install the same package with
    
    
    sudo pacman -S firefox
