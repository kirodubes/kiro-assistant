# Installing minecraft on any Arch Linux based system

![](https://arcolinux.com/wp-content/uploads/2019/07/minecraft.jpg)

Installing minecraft was not much of a challenge really but then we did not set up a server or where talking about open up ports on your router.

So this is just a local installation to play minecraft and connect to someone else's server providing the ports are open.

If you want to try out something new we first google.

arch wiki minecraft

The result is this link :

<https://wiki.archlinux.org/index.php/Minecraft>

We read the text and follow the steps in the video.
    
    
    yay -S minecraft-launcher

Then we need a login and a password and we are playing.

It seems to be better to launch minecraft from the menu

There is a full screen option in the options
