# How to check if the downloaded iso is not corrupted

![](https://arcolinux.com/wp-content/uploads/2020/01/md5-checks.jpg)

We follow the advice posted on the download page.

<https://arcolinux.info/download/>

We will update the terminal codes only there to keep it manageable.

 

**Why** **would you want to check the integrity of the iso?**

If you want to **check** if**your download** is not **corrupted** in any way, you can do so.

> Basically you check if the iso we have is the same iso you have. Bit by bit.

Depending **where** you download, you will have the **sha1** , **sha256** and **md5sum** of each iso. You will also receive a **.sig** file to see if Erik Dubois has signed that file.
