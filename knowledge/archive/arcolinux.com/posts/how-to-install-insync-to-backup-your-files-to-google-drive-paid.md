# How to install insync to backup your files to google drive - paid

![](https://arcolinux.com/wp-content/uploads/2018/04/arcolinux-insync.jpg)   


Working on any operating system you need to have a **contingency** **plan**.

> What if my systems breaks down for whatever reason. Old age for example.

Use the free cloud services at hand like dropbox or ...

[Insync is a sync tool for google drive BUT it is **not free**](<https://www.insynchq.com>).

We see that we can install it from the AUR or Arch User Repository.
    
    
    trizen insync

After installing the application you need to login or make an google account to be able to sync your documents in the folder to your google drive. I change the folder where it wants to sync to and will go over various settings of the tool.
