# Installing fun stuff for the terminal on ArcoLinux

![](https://arcolinux.com/wp-content/uploads/2018/07/arcolinuxd-pywal-cava-ranger-pipes-yellow-1.jpg)

Use our meta package to install all these fun applications
    
    
    sudo pacman -S arcolinux-meta-fun

What will be installed :

  1. asciiquarium
  2. bashtop
  3. cmatrix
  4. cool-retro-term
  5. cowfortune
  6. figlet
  7. lolcat
  8. mc
  9. ranger
  10. sl
  11. ufetch-arco
  12. bash-pipes
  13. boxes
  14. cava
  15. cpufetch-git
  16. curseradio-git
  17. cxxmatrix-git
  18. gotop-bin
  19. python-pywal
  20. toilet
  21. slurm
  22. tty-clock
  23. ufetch
  24. pfetch
  25. nixieclock
