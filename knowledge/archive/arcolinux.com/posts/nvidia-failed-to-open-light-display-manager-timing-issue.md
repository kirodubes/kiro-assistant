# Nvidia - Failed to open Light Display Manager - timing issue

![](https://arcolinux.com/wp-content/uploads/2021/01/failed.png)

One of the reasons of this error message

> Failed to open light display manager

is that the kernel (in this case linux) and the Nvidia packages were not downloaded/synced at the same time.

## Solution

Wait or find another server and update via TTY.

Then reboot.

Video: <https://youtu.be/KtScvhWIa5I>
