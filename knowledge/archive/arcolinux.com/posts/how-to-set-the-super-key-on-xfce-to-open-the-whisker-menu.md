# How  to set the super key on XFCE to open the whisker menu

![](https://arcolinux.com/wp-content/uploads/2019/07/whiskermenu-xfce.jpg)

Do not do this on the ArcoLinux iso   
(Xfce, Openbox and i3)

Setting the keyboard shortcut  
will cause conflict in Openbox

Great tip for  
ArcoLinuxD  
ArcoLinuxB

We have left out the keyboard shortcut to open the (whisker) menu when pressing the super (windows) key because of the conflict it will create in Openbox. The ArcoLinux iso is the template for other desktops so they will have the same setting.

You can get it back simply by creating a keyboard shortcut (super) to the application **xfce4-popup-whiskermenu**.

See in the video how it is done.

Video: <https://youtu.be/r2qH_IQ5YB0>
