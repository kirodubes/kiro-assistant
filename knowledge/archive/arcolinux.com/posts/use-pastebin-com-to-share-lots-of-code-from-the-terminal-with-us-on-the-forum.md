# Use pastebin.com to share lots of code from the terminal with us on the forum

![](https://arcolinux.com/wp-content/uploads/2019/05/pastebin.jpg)The idea of this help menu is that we need to be able to help you. 

> In order to be able to help you, we need information.

Provide us with the output in your terminal. Select the output code in termite or urxvt. Press **CTRL + SHIFT + C.** It is now copied. Go online to <https://pastebin.com>. Make on account there or do not make an account. Copy/paste the code in the textbox with **CTRL + V**. Follow the next steps to save this code and get the url. The url can be found in the textbox of your browser. Copy/paste that url on our forum.
