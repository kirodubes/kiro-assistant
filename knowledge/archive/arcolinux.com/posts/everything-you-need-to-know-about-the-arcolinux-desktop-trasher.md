# Everything you need to know about the ArcoLinux Desktop Trasher

![](https://arcolinux.com/wp-content/uploads/2021/09/arcolinux-desktop-trasher.png)

The ArcoLinux Desktop Trasher is created to give you the possibility to remove any desktop from your system.

The ArcoLinux Tweak Tool will install it.

The ArcoLinux Desktop Trasher will remove it.

We added a switch not to touch your .config folder. That is activated by default.

You can also follow what packages were installed or removed via the log files in **/var/log/arcolinux**.
