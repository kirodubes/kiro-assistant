# Installing the nvidia driver for GEFORCE GTX970 via pacman - November 2019

![](https://arcolinux.com/wp-content/uploads/2019/10/nvidia-inxi.jpg)

Once I knew what package I needed to install for my

**GEFORCE GTX970**

it was smooth sailing all the way.
    
    
    sudo pacman -S nvidia nvidia-settings

Then you reboot and hope for the best like we all do.

In my case that is all it took.
