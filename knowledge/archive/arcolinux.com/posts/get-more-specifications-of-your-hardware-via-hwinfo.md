# Get more specifications of your hardware via hwinfo

![](https://arcolinux.com/wp-content/uploads/2018/09/arcolinuxb-bspwm-hwinfo.jpg)   


The application hwinfo provides you (very fast I may add) with all the technical information you could desire.

It may be already installed on your version otherwise install it with
    
    
    sudo pacman -S hwinfo

To be able to see the short version from the video, try this in a terminal
    
    
    hwinfo --short

You can do so much more.

Help yourself by reading the help.
    
    
    hwinfo --help

More info about your bios would be
    
    
    hwinfo --bios

Depending on your version you will find an alias in .bashrc that shows you the short list. The alias is 
    
    
    hw
