# Installing and playing neverball - coordination game

![](https://arcolinux.com/wp-content/uploads/2018/11/neverball.jpg)

Neverball is hand coordinating game. You see me play 2 levels (the easiest ones).

It will get a lot more difficult.

Check out the [screenshots](<https://neverball.org/screenshots.php>) on the official website here.

Install it with
    
    
    sudo pacman -S neverball
