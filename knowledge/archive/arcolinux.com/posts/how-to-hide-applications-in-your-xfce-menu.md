# How to hide applications in your Xfce menu

![](https://arcolinux.com/wp-content/uploads/2018/09/hide-applications-in-menu-xfce.jpg)

We open the **menulibre** applications via the whiskermenu and select the applications we do not want to see and choose hide from menu's.

That is it.

A file is being created in ~/.local/share/applications as you can see in the video.

Video: <https://www.youtube.com/watch?v=ejwaoKFzTd0>
