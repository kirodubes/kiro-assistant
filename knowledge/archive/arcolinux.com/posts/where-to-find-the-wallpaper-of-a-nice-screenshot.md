# Where to find the wallpaper of a nice screenshot

![](https://arcolinux.com/wp-content/uploads/2019/05/arcolinuxb-qtile-tineye.jpg) People always ask me where can I find this or that wallpaper. When making tutorials [variety](<https://arcolinux.com/?s=variety>) is providing new wallpapers all the time. I never go look for a wallpaper. If there is a wallpaper I like, I will save it on my account at https://desktoppr.co/erikdubois. As a result it will be saved on my dropbox that is linked to that website. So if you see a wallpaper you like, see if I have saved it on desktoppr. **Another way** **to find the wallpaper in question** , is using a **reverse image search** like <https://www.tineye.com> Upload the screenshot to this website and select the best possible resolution for your computer. 

## Extra tip:
