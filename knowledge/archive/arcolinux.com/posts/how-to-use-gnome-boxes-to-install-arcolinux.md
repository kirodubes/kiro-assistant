# How to use gnome-boxes to install ArcoLinux

![](https://arcolinux.com/wp-content/uploads/2020/09/boxes.jpg)

Installing gnome-boxes could not be any simpler. Open a terminal and type
    
    
    sudo pacman -S gnome-boxes

Now reboot.
