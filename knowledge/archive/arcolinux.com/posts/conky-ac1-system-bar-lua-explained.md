# Conky AC1-System-Bar-LUA explained

![](https://arcolinux.com/wp-content/uploads/2019/07/ac1-system-bar-purple.jpg)![](https://arcolinux.com/wp-content/uploads/2019/07/ac1-system-bar.jpg)

Any conky with the word **LUA** in the name is going to rely on the language lua and the package we install standard **conky-lua-archers**.

So this conky is not the regular conky - it is written in the language LUA.

We see where the conky is and how it is started but the real conky is in a LUA file - system-bar.lua.

We show you the code in **Sublime-text**. You can also open it in **Atom**. Choose the editor you like but see that you get **syntax** support as well.

We change the color theme and show some coloring possibilities.

You can make the **black** background a **transparent** one.

Experiment with all the colors and the settings and make your self a nice desktop.
