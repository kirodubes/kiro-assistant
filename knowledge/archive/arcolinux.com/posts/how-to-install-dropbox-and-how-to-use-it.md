# How to install dropbox and how to use it

![](https://arcolinux.com/wp-content/uploads/2017/10/dropbox.jpg)   
Working on any operating system you need to have a contingency plan. What if my systems breaks down for whatever reason. Old age for example. Use the free cloud services at hand. Dropbox is a cloud service. Any project we start, begins in [Arch Wiki](<https://wiki.archlinux.org/index.php/dropbox>). We see that we can install it from the AUR or Arch User Repository. 
    
    
    yay -S dropbox

After installing the application you need to login or make an account to be able to sync your documents in the folder ~/dropbox to the cloud.   
  


Keyboard Shortcuts To Remember

SUPER + T ... CTRL + ALT+ T ... Super + Enter
