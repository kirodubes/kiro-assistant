# Adding jgmenu to your polybar

![](https://arcolinux.com/wp-content/uploads/2018/07/arcolinux-openbox-polybar-jgmenu.jpg) In this [article](<https://arcolinux.com/adding-jgmenu-to-your-openbox-system/>) we installed **jgmenu** and talked about this new menu called **jgmenu** coming from this [github](<https://github.com/johanmalm/jgmenu>). We can add this menu as well to polybar. [Polybar](<https://github.com/jaagr/polybar>) is an alternative menu you can use on Openbox, i3, Bspwm and so many more. We will now show you how to get this jgmenu in your polbar on Openbox. With that knowledge you can do it on i3 and Bspwm later as well. Install jgmenu and init. install polybar and use all cores to install faster. Change from tint2 to polybar in Openbox. Add jgmenu to Polybar config - Openbox mainbar. Logout - Login - Enjoy Copy/paste our config of jgmenu from [pastebin.com](<https://pastebin.com/jcQMhxSF>)
