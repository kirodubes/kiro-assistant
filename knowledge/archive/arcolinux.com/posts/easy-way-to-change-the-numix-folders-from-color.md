# Easy way to change the numix folders from color

![](https://arcolinux.com/wp-content/uploads/2018/07/numix-folders-blue.jpg)![](https://arcolinux.com/wp-content/uploads/2018/07/numix-folder-standard.jpg) There is an application that will let you change ONLY the Numix folders from color AND choose also a different kind of folder if you want it. Just point and click and apply. Could not be simplier. It will overwrite the original Numix folder icons. But you can reset it later to the standard colors. 
    
    
    trizen numix-folders-git

Make sure you have the Numix icons installed. ![](https://arcolinux.com/wp-content/uploads/2018/07/numixfolders.gif)
