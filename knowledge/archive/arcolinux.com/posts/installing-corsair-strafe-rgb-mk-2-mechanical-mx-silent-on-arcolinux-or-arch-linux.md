# Installing Corsair Strafe RGB MK.2 Mechanical MX Silent on ArcoLinux or Arch Linux

![](https://arcolinux.com/wp-content/uploads/2019/07/Corsair-STRAFE-RGB-MK2.jpeg)

Corsair just happens to be the predominant keyboard my pc shop sells. 

I want people to know that this particular keyboard works and that you can even play around with all the keyboard settings.

ArcoLinux and Arch Linux approved.

Installation script is on https://github.com/erikdubois/arcolinux-nemesis - like so many other interesting scripts.

This one is easy to install
    
    
    yay -S ckb-next-git  
    sudo systemctl enable ckb-next-daemon.service

Now reboot.

> Less noice when typing - silent mechanical keys
