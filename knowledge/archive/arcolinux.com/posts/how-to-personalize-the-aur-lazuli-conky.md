# How to personalize the AUR-Lazuli conky

![](https://arcolinux.com/wp-content/uploads/2018/11/lazuli-conky-azul.jpg)

After the article how to create [AUR-Azuli and set its vnstat service](<https://arcolinux.com/the-creation-of-aur-lazuli-conky-and-configuration-of-vnstat/>) we can now change the conky to our own preferences.

We copy/paste the conky so that it will never be overwritten when you do **skel and give it a unique name.**

We change : 

  * use gpick to get some color swatches
  * colors of the text
  * size of the font
  * if no font defined, the general font and size kicks in
  * we delete more memory lines
  * we delete more cpu lines
  * we compare with meld what we changed
