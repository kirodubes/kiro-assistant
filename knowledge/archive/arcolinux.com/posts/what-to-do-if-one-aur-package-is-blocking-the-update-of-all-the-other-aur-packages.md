# What to do if one AUR package is blocking the update of all the other AUR packages

![](https://arcolinux.com/wp-content/uploads/2019/01/aur-updates-blocking.jpg)

In this video there are a lot of updates coming in from AUR. One of them has an error in the build and that stops all the updates in its tracks.

With **pamac-aur** we can easily overcome this and in a graphical environment.

Make sure you add the AUR to the settings of pamac to do so. See image below. Go to **preferences.**

![](https://arcolinux.com/wp-content/uploads/2019/01/aur-pamac.png)

Video: <https://www.youtube.com/watch?v=izsWWKcTRD4>
