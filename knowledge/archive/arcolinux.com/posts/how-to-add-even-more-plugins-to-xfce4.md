# How to add even more plugins to Xfce4

![](https://arcolinux.com/wp-content/uploads/2018/12/xfce4-plugins.jpg)

When you install Xfce and follow the [Arch wiki guide](<https://wiki.archlinux.org/index.php/xfce>) you can install the two main packages

  * xfce4
  * xfce4-goodies



[Xfce4-goodies](<https://www.archlinux.org/groups/x86_64/xfce4-goodies/>) contains already a LOT of plugins. These plugins will work and are coming from the Arch repositories.

You can add them to your xfce4 panel. 

Right mouse click on the panel will let you add them.

But are there more plugins.

On Arch Linux / ArcoLinux we will always have the AUR... so see what you can find on the AUR via the terminal and select the number that interests you
