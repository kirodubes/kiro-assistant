# Provide us with technical data of your computer with probe so we can assist you

![](https://arcolinux.com/wp-content/uploads/2021/08/probe.jpg)

Every single assist should start with

Please provide us with the probe link

At ArcoLinux we like to assist you but only if we have all the facts - all the technical data.

If you type 'probe' in a terminal you will receive a link. That link will give us all the data we need.

See the example above.

<https://linux-hardware.org/?probe=bff602ded6>
