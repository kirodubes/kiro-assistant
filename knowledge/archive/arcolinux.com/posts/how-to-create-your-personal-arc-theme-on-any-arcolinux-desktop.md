ArcoLinux has a package called **arcolinux-arc-themes-git**. It provides **45 Arc themes** not with the blue accent color but with a different accent color. These themes have been created with a script named Arc-Theme-Colora hosted at <https://github.com/erikdubois/Arc-Theme-Colora>. We build upon the work of Horst3180 at [https://github.com/horst3180.](<https://github.com/horst3180/>) **Goal of script is**

  * change blue accent into one of the 16 million colors
  * distro independent
  * easy to edit and change the scripts
  * follow the workflow from Horst3180
  * cutting-edge up-to-date - straight from the github

**Arc** is a flat theme with transparent elements for GTK 3, GTK 2 and Gnome-Shell which supports GTK 3 and GTK 2 based desktop environments like Gnome, Unity, Budgie, Pantheon, XFCE, Mate, etc. The script makes Arc Theme compatible with **Openbox** too.
