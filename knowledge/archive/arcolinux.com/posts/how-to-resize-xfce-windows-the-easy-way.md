# How to resize Xfce windows the easy way

![](https://arcolinux.com/wp-content/uploads/2018/06/arcolinux-xfce-tip.jpg) You can move about the windows in Xfce and put them in the corner (1/4th) or to the left or right (1/2th) or full screen. Resizing the windows can be done when going to the edges of the window and drag it into a direction. But also with the **ALT** key pressed together with the **right mouse key** **pressed** and dragging it into a direction. It will be clearer in the video.
