# Quick and easy way to get an awesome grub start screen

Since the images in Virtualbox are too small I have taken pictures with my **smartphone**. Hence the stripes and reflection. Bear that in mind.

![](https://arcolinux.com/wp-content/uploads/2019/02/arcolinux-grub-vimix-arcolinux-dual-boot.jpg)

Depending if you are on ArcoLinuxD or ArcoLinuxB make sure you have **grub-customizer** installed.

Then install our grub package
    
    
    sudo pacman -S arcolinux-grub-theme-vimix-git

Select vimix in the grub-customizer as theme.

![](https://arcolinux.com/wp-content/uploads/2019/02/grub-customizer.jpg)

Then you can reboot and enjoy the standard background.

If you want to change the wallpaper then there are two ways to do that.

  1. copy paste a wallpaper over on background.jpeg
  2. use grubcustomizer



Follow the video to know how that is done
