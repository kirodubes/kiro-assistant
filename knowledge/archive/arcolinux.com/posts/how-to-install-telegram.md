# How to install telegram

![](https://arcolinux.com/wp-content/uploads/2019/12/telegram.jpg)

Telegram is one way of communicating with ArcoLinux users and the ArcoLinux Team.  
Discord is another communication tool.

> Stay on top of things and get information from ArcoLinux all the time.

Installing **telegram** is super easy.

Open a terminal and type
    
    
    sudo pacman -S telegram-desktop

Then make an account with your mobile number.

Follow the telegram link you can find on this page
