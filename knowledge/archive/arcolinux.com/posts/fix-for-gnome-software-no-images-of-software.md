# Fix for gnome-software - no images of software

![](https://arcolinux.com/wp-content/uploads/2018/04/gnome-software-working-1.jpg) May 2018 we discovered that gnome-software no longer displayed all the images of the software. ![](https://arcolinux.com/wp-content/uploads/2018/04/ArcoLinux-gnome-software-wrong.jpg) **As of May 2018** you need these two packages to have a working gnome-software. 
    
    
    sudo pacman -S gnome-software
    sudo pacman -S gnome-software-packagekit-plugin
