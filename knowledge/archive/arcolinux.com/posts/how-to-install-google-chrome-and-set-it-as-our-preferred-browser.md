# How to install google chrome and set it as our preferred browser

![](https://arcolinux.com/wp-content/uploads/2018/01/archmerge-google-chrome.jpg)   We learn how to install **Google Chrome**. It is coming from the AUR. So you can use [many AUR helpers out there](<https://wiki.archlinux.org/index.php/AUR_helpers>). Then we find out how the application is called. You can do this via pacman as well but we also have a "**dmenu** " installed. Pressing **Super + Shift +D** activates it and in 1 second we know what the command is. Then we set it in the **Preferred Applications**.
