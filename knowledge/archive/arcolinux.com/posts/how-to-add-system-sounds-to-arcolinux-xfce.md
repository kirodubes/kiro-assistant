# How to add system sounds to ArcoLinux Xfce

![](https://arcolinux.com/wp-content/uploads/2018/03/xfce-system-sounds.jpg)   


If you like you can install system sounds to your XFCE desktop. It is actually not that difficult.

Everything you then do like clicking start icon, close windows, clean the trash will make a sound. I am not a fan of all these noices that will surely not improve my music experience.

As a result we will not include this on the iso but we will show you how to do it for educational purposes.

We install a sound package. There are many many sound packages out there.
    
    
    packer sound-theme-smooth

We use pamac to investigate where are all the files are.

We enable **event sounds** in Xfce in Appearance.

In the settings editor we can change a lot of settings.

We changed the **SoundThemeName** from **default** to **Smooth**.

Now you can enjoy system sounds on your Xfce.
