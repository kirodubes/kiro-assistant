# How to theme polybar to match an orange wallpaper

![](https://arcolinux.com/wp-content/uploads/2018/05/arcolinuxd-neofetch-image.jpg) Polybar can be used in many desktops. We have already visited : 

  * Openbox
  * i3
  * Bspwm

We will need it for other desktop environments we will discover. It can be themed anyway you want. It takes some time to get used to but it can be fun. We learn how to search in the config file and learn the workflow to change the settings.
