# Theming ArcoLinux when a red wallpaper comes in

![](https://arcolinux.com/wp-content/uploads/2019/06/allredtheming.jpg)

When a red wallpaper comes in, we start theming ArcoLinux all red.

Conky logo is chosen.

A new **combination** of **icons** is **created**. Use the inherit in your icon theme.

A new theme is chosen.

We explain the **derivatives** from Sardi. Just one icon did not change all red.

Video: <https://youtu.be/JI3s54Dk1t4>

Then we run the scripts to make the Sardi Extra and update the version to 9.7.2.

Token icon is now fixed.
