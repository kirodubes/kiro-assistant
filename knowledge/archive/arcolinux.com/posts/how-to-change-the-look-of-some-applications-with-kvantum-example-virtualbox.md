# How to change the look of some applications with kvantum example virtualbox

![](https://arcolinux.com/wp-content/uploads/2018/12/virtualbox-before-update.jpg)![](https://arcolinux.com/wp-content/uploads/2018/12/virtualbox-updates-kernel.jpg)![](https://arcolinux.com/wp-content/uploads/2018/12/virtualbox-after-update.jpg)

31 December 2018 we get an update in : **new kernel 4.20 and virtualbox 6.0.**

The first image is virtualbox before the update.

The second image is the content of the update.

The third image is virtualbox after the update.

Time to do something about this "new look" of Virtualbox 6.0.

We install the following packages.
    
    
    sudo pacman -S kvantum-qt5 kvantum-theme-arc

Then we choose the theme we like with two applications

**qt5ct and kvantummanager.**
