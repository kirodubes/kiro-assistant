# Where do I set my preferred applications in Xfce

![](https://arcolinux.com/wp-content/uploads/2018/04/arcolinux-preferred-applications.jpg) You can set your preferred applications from the menu. Only for 

  * browsers
  * mail reader
  * file manager
  * termite

This application we start, is actually **exo-preferred-applications**. ![](https://arcolinux.com/wp-content/uploads/2018/04/arcolinux-preferred-applications-0.jpg)![](https://arcolinux.com/wp-content/uploads/2018/04/arcolinux-preferred-applications-1.jpg)
