# Install bash-pipes to have a colorful screensaver in your pimped terminal

![](https://arcolinux.com/wp-content/uploads/2018/05/arcolinux-bash-pipes.jpg) You often see these images or also called scrots (see the application "scrot") with colored pipes or lines. You can have those too if you install: 
    
    
    trizen bash-pipes

Then you better learn the options with 
    
    
    pipes --help

and have fun trying out the possibilities.
