# Everything you need to know about snap

![](https://arcolinux.com/wp-content/uploads/2020/11/trackmania.jpg)![](https://arcolinux.com/wp-content/uploads/2020/11/strimio.jpg)

[Snapcraft](<https://snapcraft.io>) from Canonical wants to provide the app store for Linux.

Not all packages come from Arch Linux or AUR.

There will always be packages that are (not yet) on AUR.

In that case you can see if there is a snap package.

[Do check out the Arch Linux Wiki page to get more info on snap.](<https://wiki.archlinux.org/index.php/Snap>)

This tutorial is created by coincidence. A user on Telegram mentioned the application **strimio**. So we see if we can install it on ArcoLinux.

There is **NO** **PKGBUILD** on **AUR** for **strimio** (at this time).

Navigating to the website we learn we can install it via **snap** **store**.

The application strimio as such is unimportant to me. My interest lies in the technical part.

Can it be done?  
What do I need to do?
