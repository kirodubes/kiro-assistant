# How to burn a Windows 10 iso to an usb on Linux

![](https://arcolinux.com/wp-content/uploads/2018/06/w10-woeusb.png) Sometimes you want to burn a Windows iso on an usb to test out this operating system on bare metal and not in a virtual machine. The application **woeusb** will help you achieve this. More info about the application on the [github](<https://github.com/slacka/WoeUSB>). In the video we will use the gui or graphical user interface of this application. It could not be simpler. We install it using trizen (we choose the git version) 
    
    
    trizen woeusb-git
