# How to fix the system panel icons in i3

![](https://arcolinux.com/wp-content/uploads/2018/12/i3-fix-icons.jpg)At this point in time I can see just 2 options. Install the package **gtk3-mushrooms** coming from AUR. Or get rid of the system panel icons altogether. The reasoning is that we can reach those applications if we want to via shortcuts or terminal. We will show you the last solution in this video. You can get rid of the system icons with this line of code 
    
    
    #system icons
    tray_output none
