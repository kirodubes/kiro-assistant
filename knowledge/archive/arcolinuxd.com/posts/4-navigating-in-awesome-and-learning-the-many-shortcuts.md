# 4 Navigating in Awesome and learning the many shortcuts

![](https://www.arcolinuxd.com/wp-content/uploads/2018/03/arcolinux-awesome-keyboard-shortcuts.jpg)

Navigating around in a windows tiling manager is different than in other windows managers.

We have tagnames to put applications on and we learn the most important shortcuts.

There is a very important shortcut to set your tiling and that is **Super + Spacebar.**

**Super + esc** is a supertip to **kill** a process that you want to stop **immediately**.

**Rofi** has been added to Awesome :

Super + R to select menu - ALT + A to save it

Super + F11 to choose menu's

Super + F12 to choose menu's

You can add **gaps** in Awesome between windows with shortcuts (CTRL + ALT +M and P) but you can set it also in the theme.lua. Change the value in **theme.useless_gap**.
