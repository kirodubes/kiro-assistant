install desktop environment 

Gnome

![](https://www.arcolinuxd.com/wp-content/uploads/2018/10/archlinux-gnome-clean-install.jpg)![](https://www.arcolinuxd.com/wp-content/uploads/2018/10/archlinux-gnome-cleaninstallapps.jpg)

Installing Gnome requires only a few lines. Theming and tweaking your Arch Linux system will require more time. All the information about this desktop is here:

<https://wiki.archlinux.org/index.php/gnome>
    
    
    sudo pacman -S gnome gnome-extra

Video: <https://youtu.be/fpw1BwpKiC4>

Your installation can just end here. Then you add more applications to have a system suited to your needs and theme it anyway you like.
