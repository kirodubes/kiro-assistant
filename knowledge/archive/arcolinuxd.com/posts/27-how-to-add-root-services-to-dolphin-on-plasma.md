# 27 How to add root services to dolphin on plasma

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/root-actions.jpeg)

On Xfce and other desktops it is possible to "take actions with root level".

That is possible too on Plasma BUT you need to install a service for that.
    
    
    yay -S kde-servicemenus-rootactions

This package will come from the [AUR](<https://aur.archlinux.org/packages/kde-servicemenus-rootactions/>) and its origin is on [store.kdg.org](<https://store.kde.org/p/998469>).   
The developer gives you updates on this website.   
You can follow the changelog there during his development.

Check the image above to see all the possible options.

You can install it via yay but you can install it also via dolphin.

You can install it via **yay** but you can install it also via **dolphin**.

Go to Control - Configure Dolphin - Services - Download new services and search for the extra functionality you want.

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/root-services-via-dolphin.jpeg)

Video: <https://youtu.be/GwwwBPwHbXw>
