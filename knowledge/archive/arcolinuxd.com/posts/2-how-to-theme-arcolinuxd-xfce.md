# 2 - How to theme ArcoLinuxD Xfce

ArcoLinuxD is installed and we ran all the scripts from the folder arco-xfce. Then we tweak our distro.

**Settings/Appearance** is where we can set themes, icons and fonts.

**Settings/Window manager** is where we set our border of our windows and fonts.

**Settings/Mouse and Touchpad** is where we set our cursor.

We use **variety** to give us a better wallpaper and delete one of them so we do not see it again.
