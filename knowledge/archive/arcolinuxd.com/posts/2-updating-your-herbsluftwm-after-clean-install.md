# 2 updating your Herbsluftwm after clean install

![](https://www.arcolinuxd.com/wp-content/uploads/2019/10/hlwm-update-system.jpg)

You can install Herbstluftwm via ArcoLinuxD and use and change the scripts we provide.

You can also install Herbstluftwm via ArcoLinuxB and download the ISO and install it or you can build the ISO yourself and decide what software you would like on your ISO.

In the end you will always need to update your system.

In the video we use the aliases **update** , **skel** , **cb,** **upall (****upall)** and **bupskel**.

There is no need to rehash this knowledge here. You can find all the information you require here :
