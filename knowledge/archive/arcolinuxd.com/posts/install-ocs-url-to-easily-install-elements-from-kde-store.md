# Install ocs-url to easily install elements from kde store

![](https://www.arcolinuxd.com/wp-content/uploads/2020/01/ocs-url.jpg)

You can install elements from the [KDE store](<https://store.kde.org/>) with a tool called [ocs-url](<https://www.pling.com/p/1136805/>). You can also install them via your system settings and download them from there.

I do like the idea going over the KDE store and then choose the things I like after seeing some images in their full size.

Ocs-url can be easily installed with
    
    
    yay -S ocs-url

Check out the video how to install it and how to use it.
