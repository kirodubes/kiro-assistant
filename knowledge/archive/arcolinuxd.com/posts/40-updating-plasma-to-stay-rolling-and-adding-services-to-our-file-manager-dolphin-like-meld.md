# 40 Updating Plasma to stay rolling and adding services to our file manager dolphin like meld

![](https://www.arcolinuxd.com/wp-content/uploads/2018/12/arcolinux-plasma-update.jpeg)

Updating is always necessary to keep rolling. Since this is a 18.11 version I needed to update to a 18.12 version. 

There are other video's that explain that as well on ArcoLinux and other ISO's. [You can see it in the release article as well](<https://arcolinux.info/arcolinux-d-b-18-12/>).

**But the point of the video was actually to show you how and where to add services into dolphin.**

When we are on XFCE or work with thunar in general we can compare two files with meld because we added a custom action.

Can we have the same on Dolphin?

Yes.

We install several **dolphin** **extensions** in the video.

\- compare files with meld  
\- convert png's to jpg's  
\- make a file executable

**Mind you these extensions are written by users and will not always work.**

Video: <https://youtu.be/C3VEGATbvTM>
