# 18 install lastpass with yay and sync your firefox after a clean install - any desktop

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/firefox-sync-lastpass.jpeg)

One of my goals is to be up and running as fast as possible on ANY DESKTOP of ArcoLinux.

Syncing your settings, extensions and bookmarks to your firefox cloud account can be an asset in that matter.

All I need to do is install the extension lastpass and login to my firefox account and everything syncs again.
    
    
    yay -S lastpass

Video: <https://youtu.be/e2VyKqRblPc>
