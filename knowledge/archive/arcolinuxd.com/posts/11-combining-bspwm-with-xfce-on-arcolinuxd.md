# 11 Combining Bspwm with Xfce on ArcoLinuxD

![](https://www.arcolinuxd.com/wp-content/uploads/2019/01/arcolinuxd-xfce-bspwm.jpg)

First we install **ArcoLinuxD** on virtualbox or on real metal.

The next video will show you that.

Video: <https://youtu.be/F-FA5swPiX4>

After the installation of **ArcoLinuxD** on virtualbox or on real metal we reboot.

Immediately we check if we need to do updates but first we get the fastest servers for ArcoLinux in.

**mirror** or **mirrors** or **mirrora** or **mirrord**

then
    
    
    update
    upall

Then we start installing our choice of desktops.

This video combines **Xfce4** and **Bspwm** but could have been any other tiling window manager for example.

We get the scripts from <https://github.com/arcolinuxd/>. 

ALLWAYS RUN SCRIPT 400, 500, 600  
DISTRO SPECIFIC APPLICATIONS  
ARCOLINUX CONFIGURATIONS

script 600 does skel by default
