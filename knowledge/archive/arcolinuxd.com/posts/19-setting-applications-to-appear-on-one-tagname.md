# 19 Setting applications to appear on one tagname

![](https://www.arcolinuxd.com/wp-content/uploads/2018/03/arcolinux-awesome-gimp.jpg)

We want to force Awesome to open an applicaton on a specific tagname and a specific screen. Dual screen users can use these settings. Or Multi screen users for that matter.

As example we want to launch Atom always on screen 1 and tag 2. We apply this and test it out.
