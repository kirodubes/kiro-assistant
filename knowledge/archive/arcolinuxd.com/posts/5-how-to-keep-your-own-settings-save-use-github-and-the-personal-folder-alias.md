# 5 How to keep your own settings safe - use github and the personal folder/alias

![](https://www.arcolinuxd.com/wp-content/uploads/2021/10/leftwm-theme.jpg)

After we changed the ArcoLinux Leftwm theme called Candy we want to keep our theme.

We copy/paste the folder and rename it to brandy. The folder brandy will never be overwritten. Then later we tell leftwm to change the theme from candy to **brandy**.

Keep your data safe on github. [Here is the tutorial](<https://www.arcolinuxd.com/easy-way-to-learn-about-git-and-github/>).

You can learn about the [Personal folder and the personal alias](<https://arcolinuxb.com/personal/>) and have fun with that.

We explain how skel works. [Still not clear read this](<https://arcolinux.com/updating-your-system-with-skel-and-still-keeping-your-own-settings/>).
