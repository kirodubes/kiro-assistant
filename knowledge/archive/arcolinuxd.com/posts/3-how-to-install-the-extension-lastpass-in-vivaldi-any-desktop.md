# 3 How to install the extension lastpass in Vivaldi - any desktop

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/lastpass.jpeg)

In order for me to be able to upload to my youtube channel I need to have my login and password. They are stored safely online with the tool from [lastpass.com](<https://www.lastpass.com>). We launch Vivaldi and show you how to install the extension **lastpass**. It comes from the Google store. You can also install it via AUR with:
    
    
    yay -S lastpass

There are other alternatives out there. I have no affiliation with lastpass. Just a satisfied customer. Google for **lastpass alternatives** and you will get a wide selection of other applications that you can explore.

Video: <https://youtu.be/qomdFbXwpPI>
