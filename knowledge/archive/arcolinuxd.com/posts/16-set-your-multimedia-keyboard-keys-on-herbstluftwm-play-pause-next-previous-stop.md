# 16 Set your multimedia keyboard keys on Herbstluftwm - play pause next previous stop

![](https://www.arcolinuxd.com/wp-content/uploads/2019/10/hlwm-music.jpg)

We have installed **volumeicon** in this tiling window manager.

You can use/install it in any tiling window manager. As always that is the choice for the user.

Personally I use **pavucontrol** as hub to set all my sound the way I need it to be. CTRL + ALT + U

**If you want to use the keyboard** shortcuts, you have to set them in **sxhkd** config because you can use both systems

  * mpd
  * playerctl



Playerctl is used by spotify for example. Always check out any Arch Wiki if you get stuck - [see spotify arch wiki](<https://wiki.archlinux.org/index.php/Spotify>).

[Mpd](<https://wiki.archlinux.org/index.php/Music_Player_Daemon>) is used by [ncmpcpp](<https://wiki.archlinux.org/index.php/Ncmpcpp>).
