# 7 - Installing software with pamac or gnome-software any desktop

In ArcoLinuxD we install all our software with scripts.

Scripts can be easily edited. Delete lines, add lines. Install the software YOU like. That must be the first step to install the applications you do want.

But eventually you will want to try out new applications you see on the net.

There are two applications you can use to install and uninstall applications. I will show you both in this tutorial.

**pamac and gnome-software**

Both are very efficient and eye-candy.

Check it out.
