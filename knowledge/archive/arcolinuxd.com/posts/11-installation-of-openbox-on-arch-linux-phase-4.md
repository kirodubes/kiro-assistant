# Installation of Openbox on Arch Linux Phase 4

install desktop environment OPENBOX ![](https://www.arcolinuxd.com/wp-content/uploads/2018/04/archlinux-openbox.jpg) Installing openbox requires only a few lines. Theming and tweaking your Arch Linux system will require more time. All the information about this desktop is here: <https://wiki.archlinux.org/index.php/openbox>
    
    
    sudo pacman -S openbox

Video: <https://youtu.be/ICDHnH41urI>

Let us go over every settings of Arch Linux Openbox and see what we still need to fix.

We get the .**bashrc** from ArcoLinux to get **upall** and **mirror** for example.

**sudo pacman -S reflector** — You need to install **reflector** to use mirror.

Then we install **trizen with yaourt**.

Then we install **spotify** with trizen.

We change the **conky** and change the logo of the conky.

We run **temps** and set it to brussels.

We change the theme, icons and cursor. We need to do this in every application about theming. Lxappearance, Xfce4-appearance and GUI Config Tool.

We change the theme of **termite**.

We change the tint2 theme.

We delete some of the **menu's** because they point to ArcoLinux.
