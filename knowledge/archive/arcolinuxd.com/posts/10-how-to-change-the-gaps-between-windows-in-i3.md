# 10 - how to change the gaps between windows in i3

We have choosen to take i3 next gap from [airblader](<https://github.com/Airblader/i3>). It provides you with the same standard i3 but with the possibility to have gaps between the windows. IF you do not like these gaps then you make your gaps 0 and you will have the regular i3.

Change the setting in ~/.config/i3/config at the bottom.
