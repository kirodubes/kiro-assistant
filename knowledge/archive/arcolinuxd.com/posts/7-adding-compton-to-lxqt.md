# 7 Adding compton to LXQt

![](https://www.arcolinuxd.com/wp-content/uploads/2020/04/compton-picom.jpg)

**Compton** changed its name during 2019 into **Picom**. As a result we changed the name references from compton.conf to**picom.conf**.

Update your system and use skel to get the new files in your home directory. Then you can reboot. This was anno April 2020. Next ArcoLinuxB ISOs will have these settings on board.

Video: <https://youtu.be/MGZMBCcf1Mc>

Compton/Picom can take care of 3 elements

  * shadow
  * transparency or opacity
  * fading



You will find the settings file for picom in ~/.config/lxqt/picom.conf

Let us experiment with the settings.

This applies of-course to any desktop that uses Picom.

Video: <https://youtu.be/RLgq463Ng-c>
