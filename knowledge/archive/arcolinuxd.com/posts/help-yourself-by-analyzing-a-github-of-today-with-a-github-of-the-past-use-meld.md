# Help yourself by analyzing a github of today with a github of the past - use meld

![](https://www.arcolinuxd.com/wp-content/uploads/2018/10/comparegithubs.png) Github is quite the invention from Linus Torvalds. Github can be a Time Machine. You can retrace all the changes ever made and you can even get the complete situation back at some point in time - IN THE PAST. In the video we will show you why that can be important and how it can be done. It will help you analyze your issues and see if a change in code or a change of application or just even a typo is the reason why things are different now compared with months ago.
