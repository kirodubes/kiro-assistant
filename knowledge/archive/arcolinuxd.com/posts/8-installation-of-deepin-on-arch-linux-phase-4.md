# Installation of Deepin on Arch Linux Phase 4

install desktop environment deepin ![](https://www.arcolinuxd.com/wp-content/uploads/2018/04/archlinux-deeping-screen.jpg)Installing deepin requires only a few lines. Theming and tweaking your Arch Linux system will require more time. All the information about this desktop is here: <https://wiki.archlinux.org/index.php/Deepin_Desktop_Environment>
    
    
    sudo pacman -S deepin
    sudo pacman -S deepin-extra

Video: <https://youtu.be/WheI4F66qfs>

ArcoLinuxD github is the source 

We will get our scripts from our github. Change the scripts any way you want. That is the idea. Add lines for extra software. Hashtag out software you do not want. Then run the scripts.
    
    
    sudo pacman -S git
    git clone https://github.com/arcolinuxd/arco-deepin

Then we have the scripts on our system. You decide what you run or not.

There is also a folder called **Arch Way**. This folder will contain all the scripts for people who do an Arch Linux installation.

Video: <https://youtu.be/MfpbX9R9x1E>

After some theming and tweaking you can end up with this desktop environment. 

Video: <https://www.youtube.com/watch?v=CKI6mT76Wr8>
