# 13 how to move applications from monitor 1 to monitor 2 or vice-versa

![](https://www.arcolinuxd.com/wp-content/uploads/2019/10/hlwm-dual-monitor-move-apps-to.jpg)

You must have a dual monitor setup to take part in this, obviously.

There is relevant code in autostart that you need to uncomment and other code that you need to comment.

Then you can reload the autostart.

With **Super + Shift + arrow keys** you can now move the active application to the other screen.
