# 8 - Installing software via the terminal any desktop

Installing software can be done via gui (graphical user interface) like pamac and gnome-software explained [here](<https://www.arcolinuxd.com/8-installing-software-via-the-terminal-any-desktop/>). But it can be done via terminal as well. Our Arch fun starts on the [AUR or Arch User Repository](<https://aur.archlinux.org/packages/?SB=v&SO=d&SO+=+a>). Check out what other people find great. Votes and popularity. What software is useful for **you** and install it via terminal or via the gui's.
