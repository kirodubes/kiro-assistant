# How to create your own plank theme the numixx theme as example - any desktop

![](https://www.arcolinuxd.com/wp-content/uploads/2019/02/plank-themes.jpg)

> Feb 2019 - 101 plank themes to choose from

![](https://www.arcolinuxd.com/wp-content/uploads/2019/02/plank-themes.gif)

In this video we will show you how easy it is to change any given plank theme.

We use the numixx plank theme as example.

We open it in the folder
    
    
    /usr/share/plank/themes/Numixx 

with **Atom** but open that specific folder as **root** to be able to save there and immediately see the result. We set the syntax to **PERL** to make it easier to read.

Then we start changing parameters and see what changes.

Video: <https://www.youtube.com/watch?v=Lhwk8WKJKdw>

Possible workflow when creating plank themes

In this video we will show you how easy it is to change any given plank theme and provide you with a possible workflow.

We use the **OSXLion** plank theme as example and make our copy unique by renaming the folder to **OSXLioness**.

We edit the files with **Atom** and set the syntax to **PERL** to make it easier to read.

Then we start changing parameters and see what changes.
