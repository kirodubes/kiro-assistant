# 19 How to install dropbox and safeguard all your documents in the cloud - any desktop

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/dropbox.jpeg)

Check the version of Dropbox on  
AUR and 3Party repo  
Install the most recent version

Since we use **dropbox** as a service constantly on all our installations of all the desktops we have added dropbox to the 3party repo. This version will not be auto updated. It is a manual job. One we tend to forget. It is not that important anyway. Dropbox will keep working whether it is version x or y.

The **advantage** of having a 3party repo is the **installation** **time**. It will be faster as a build from AUR.

**You decide to install it from aur or from 3party repo.**  
**Check the version number.**

sudo pacman -S dropbox
