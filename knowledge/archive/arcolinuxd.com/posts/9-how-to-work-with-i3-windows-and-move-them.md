# 9 - how to work with i3 windows and move them

Let us show you how to move your windows around.

We go over the many keyboard shortcuts.

**Have fun trying these out in a working system.**

ALT + TAB

SUPER + arrows

CTRL + ALT + arrows

SUPER + SHIFT + arrows

SUPER + SHIFT + numbers

 

SUPER + F is **full** **screen** and is a great tip.

SUPER + SHIFT + SPACEBAR is floating. Move the windows with SUPER PRESSED and the mouse.
