# 17 How to set the super key on xfce to open the whisker menu

![](https://www.arcolinuxd.com/wp-content/uploads/2019/02/whiskermenu-popup.jpg)

Pressing the LEFT SUPER key will result in opening the menu in Xfce.

This tutorial is only intended for the following situations:

  * people that will never use openbox and i3 on the ArcoLinux ISO
  * people using the ArcoLinuxD ISO and installing only Xfce
  * people using the ArcoLinuxB Xfce ISO full or minimal



> This keyboard shortcut will conflict with Openbox settings.

**Follow the tutorial to see how to set a keyboard shortcut to xfce4-popup-whiskermenu.**
