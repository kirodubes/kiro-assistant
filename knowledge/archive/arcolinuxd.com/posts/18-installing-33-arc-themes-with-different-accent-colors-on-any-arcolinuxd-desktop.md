# 18 - installing 33 Arc themes with different accent colors on any ArcoLinuxD desktop

Installing these arc themes is actually quite simple.

This package belongs to our own ArcoLinux repository and can be downloaded via
    
    
    sudo pacman -S arcolinux-arc-themes-git

I show you where on your computer this is stored i.e.
    
    
    /etc/pacman.conf

Then select your theme with **lxappearance**.
