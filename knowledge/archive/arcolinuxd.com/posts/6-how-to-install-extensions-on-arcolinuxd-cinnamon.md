# 6 - How to install extensions on ArcoLinuxD Cinnamon

Now we check out the **extensions.** By default there are **no** extensions. Let us download some of them and test them out. We test out the following extensions : 

  * Transparent panels
  * Cinnamon Maximus
  * gTile
  * Opacify windows
  * Wobbly windows
  * Center Center Panel

Remember. These extensions were build to work on Linux Mint. Some of them might not work at all.
