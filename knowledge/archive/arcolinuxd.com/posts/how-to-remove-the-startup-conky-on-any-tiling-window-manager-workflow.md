# How to remove the startup conky on any tiling window manager - workflow

Tiling window managers often start a conky to the right or to the left to help a first time user. The conky will show you the necessary keybindings to navigate in a TWM.

We show you the workflow how to remove the conky. We do not want to start it up again after every reboot.

Super + C will kill the current conky but next time you boot... it will be there again.
