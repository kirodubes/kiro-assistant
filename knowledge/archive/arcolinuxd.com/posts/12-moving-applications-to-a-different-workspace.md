# 12 moving applications to an different workspace

![](https://www.arcolinuxd.com/wp-content/uploads/2019/05/arcolinuxb-qtile-moving-windows.jpg)

**Use these keyboard shortcuts to move an application to a different workspace.**

**Super + Shift + 1 to move the focused application to workspace 1**

**Super + Shift + 2 to move the focused application to workspace 2**

...

This will work out of the box for **Qwerty** keyboard users.

If it is not working for you make sure you **DO NOT USE the numeric keypad** for the numbers and that you check if you have an **azerty** or **qwerty** or **something** **else**.

In the config you can easily switch between azerty and qwerty. Other configuration need to be researched on the net. We are happy to include it for other countries.

[In this article we change from qwerty to azerty keyboard shortcuts](<https://www.arcolinuxd.com/11-change-the-keyboard-settings-in-the-config-from-a-qwerty-to-an-azerty-keyboard-in-qtile/>).
