# 17 How to make sure the numeric keypad is enabled at boot on plasma

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/numlock-on.jpeg)

I have a keyboard with a **separate** **numeric** **keypad** and I had to manually **activate** it to be able to use the number keys.

You do this quickly manually and work on. 

But there is a **setting** in Plasma that can help you.

In the picture above and in the video you can see where we can change this.

This might not be the best thing if you have a laptop with no separate numeric keyboard pad. You decide if this is an option for you.

Video: <https://youtu.be/dkx-iKnbgk0>
