# Use tint2 to theme your Openbox setup

![](https://www.arcolinuxd.com/wp-content/uploads/2024/03/ArcoLinux_2018-03-08-42_3840x1080.jpg)

Tint2 is the "standard" panel in openbox. The gui or graphical user interface of Tint2 is a super tool to select the current themes but also a great tool to create new themes.

We change and apply some of the settings. All settings you change in the gui of tint2 will be saved in
    
    
    ~/.config/ting2/tint2rc

If you want to keep your created theme, you need to rename your own theme **but** you shoud add the **extension** : **.tint2rc**. That is what I had forgotten during the video.
