# 2 - How to theme ArcoLinuxD Cinnamon

ArcoLinuxD is installed and we ran all the scripts from the folder **arco-cinnamon**. Then we tweak our distro.

**System Settings/Themes** is where we can set themes, icons and fonts.

**System Settings/Fonts** is where we can change the font or its size.

We use **variety** to give us a better wallpaper and delete one of them so we do not see it again.
