# 6 Transforming our dash in Gnome to a dock

![](https://www.arcolinuxd.com/wp-content/uploads/2018/02/archmerged-gnome-dash-to-dock.jpg)

We miss our plank at this point in time. But gnome provides something similar a dash.

Installing plank is easy but we want to keep dash but change it to behave like plank.

We need to install a gnome shell extension :

<https://extensions.gnome.org/extension/307/dash-to-dock/>

We install the extension via the website and we show you how to add the extension of gnome to your browser.

Then we go into the settings of **Dash to Dock**.

Settings are : position, size icons, size dock, colors, transparency or opacity, move applications button to top, ....
