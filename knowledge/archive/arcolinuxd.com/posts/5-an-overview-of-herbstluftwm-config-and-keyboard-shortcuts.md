# 5 an overview of Herbstluftwm - config and keyboard shortcuts

![](https://www.arcolinuxd.com/wp-content/uploads/2019/10/hlwm-config-files.jpg)

General info

**Herbstluftwm** is a tiling window manager and it is coming from [here](<https://herbstluftwm.org/>). You can get more information on the [Arch Wiki](<https://wiki.archlinux.org/index.php/Herbstluftwm>).

The **conky** is there to help you learn the keyboard shortcuts, when you first boot into Herbstluftwm. Conkies in tiling window managers are not required nor needed.

Keep the **Arco-Herbstluftwm github** folder save for later if you have installed Herbstluftwm with the **ArcoLinuxD** ISO. In the future you can do a **git pull** and see what we improved.

We need to know the most important folders for Herbstluftwm

  * ~/.config/herbstluftwm
  * ~/.config/sxhkd



Herbstluftwm is using **polybar as menu**.

First view - keyboard shortcuts

We are using the same keyboard shortcuts on all our desktops. So learning them will benefit your workflow.

You can try out the keyboard shortcuts from the conky to start with. But there are many more. 

You need to acquire some

> finger memory

We go over some keyboard shortcuts from both configuration folders (see above).

**Super + Shift + R** is the most important one.

This keyboard shortcut will **reload** **Herbstluftwm.
