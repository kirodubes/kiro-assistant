# 3 making our terminal transparent on Virtualbox - any desktop using picom.conf

![](https://www.arcolinuxd.com/wp-content/uploads/2019/10/hlwm-termite-transparent.jpg)

Compton changed its name to picom  
compton.conf = picom.conf

ArcoLinux is always created to be perfect on real metal. Picom.conf is tweaked that way.

If you are working on Virtualbox, you can get transparency in the terminal again by putting a "#" in front of a line in the folder of herbstluftwm and other desktops using picom.

#vsync = true;
