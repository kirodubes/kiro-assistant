# 4 Syncing with Vivaldi cloud account to get all my bookmarks and more – any desktop

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/vivaldi-sync.jpg)

Like Firefox, Chromium and Google Chrome we can sync our bookmarks, settings and so much more to the cloud. 

The advantage is that after ANY CLEAN INSTALL we can get all our settings and extensions back with just a login and a password.

This way I could have installed lastpass as well. This syncing service would have downloaded the extension lastpass from the cloud. NOT the passwords mind you. Just the extension. You still need to authenticate with lastpass.com.

> If you do a lot of installations than creating a cloud account will safe you lots of time.
