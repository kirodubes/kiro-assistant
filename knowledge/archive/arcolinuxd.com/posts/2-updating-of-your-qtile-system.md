# 2 Updating of your Qtile system

![](https://www.arcolinuxd.com/wp-content/uploads/2019/05/arcolinuxb-qtile-updating.jpg)

You can install Qtile via ArcoLinuxD and use and change the scripts we provide.

You can also install Qtile via ArcoLinuxB and download the ISO and install it or you can build the ISO yourself and decide what software you would like on your ISO.

In the end you will always need to update your system.

In the video we use the commands **update** , **skel** , **upall** and **bupskel** and explain the importance of meld.

There is no need to rehash this knowledge here. You can find all the information you require here :

Update : <https://arcolinux.com/category/arcolinux/general/update/>

Compare or Meld : <https://arcolinux.com/?s=meld>
