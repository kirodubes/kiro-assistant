# 23 how to change the panel theme in Awesome and change the width of the gaps

![](https://www.arcolinuxd.com/wp-content/uploads/2019/07/arcolinuxb-awesome.jpg)

We can change the theme of the panel of Awesome that we have put at the top. You can also put it at the bottom if you like.

The themes can be changed in the file rc.lua.

The gaps need to be changed on a per theme basis. You can set every theme differently.
