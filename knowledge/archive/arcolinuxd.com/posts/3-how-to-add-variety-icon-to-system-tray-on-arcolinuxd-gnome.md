# 3 How to add variety icon to system tray on ArcoLinuxD Gnome

![](https://www.arcolinuxd.com/wp-content/uploads/2018/02/archmerged-gnome-appindicator.jpg) We changed our icons and themes with gnome-tweak-tool in previous video and installed nautilus-admin. **Gnome-extensions** can be fun but can be 'tricky' or 'unstable'. Use with care. We want to have the variety icon and we need to install a gnome extension to show the icon of variety. <https://extensions.gnome.org/extension/615/appindicator-support/> All the keyboard shortcuts for variety are present. Use them. trash wallpaper is ALT + T next wallpaper is ALT + N previous wallpaper is ALT + P etc... Check out the keyboard shortcuts in**Settings, Devices, Keyboard.**   ![](https://www.arcolinuxd.com/wp-content/uploads/2018/02/gnome-keyboardshortcuts.jpg)
