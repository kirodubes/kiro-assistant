# 5 - How to get the conky's on ArcoLinuxD Xfce

We use the scripts to install all the necessary ArcoLinux packages to get these conky's in and then we need to do some manual **copy/paste** work. 

We need to copy/paste all the conky's from the **/etc/xdg/conky** folder to our own in ~**/.config/conky**.

Then we can select the conky's with the**conkyzen**. 

Conkyzen can be found with the **right** -**mouse** -**click** IF you installed the scripts from [github in particular the one for thunar.](<https://github.com/arcolinuxd/arco-xfce>)

If you want to stop all conky's you can type the following command in the terminal
    
    
    killall conky

or go back to the conkyzen and uncheck them one by one. They will not reappear then after a reboot.
