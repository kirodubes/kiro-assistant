# 23 Using the mouse keyboard shortcuts in Qtile

![](https://www.arcolinuxd.com/wp-content/uploads/2019/05/arcolinuxb-qtile-mouse.jpg)

There are just a few mouse keyboard shortcuts and the all involve the

> SUPER KEY

So press it in and use the left mouse click and right mouse click to experiment.

If you want everything to be tiled again, reload qtile again.

SUPER + SHIFT + R

There is a keyboard shortcut to toggle between floating and non-floating

SUPER + SHIFT + SPACE
