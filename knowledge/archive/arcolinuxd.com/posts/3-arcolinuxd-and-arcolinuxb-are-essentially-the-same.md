# 3 ArcoLinuxD and ArcoLinuxB are essentially the same

![](https://www.arcolinuxd.com/wp-content/uploads/2019/02/arcolinuxb-deepin-ufetch-arco.jpg)

We just want you to know that **ArcoLinuxD** and **ArcoLinuxB** are **essentially the same**.

These articles are all hosted on ArcoLinux**D**.com.

> ArcoLinuxB delivers your operating system just way faster than ArcoLinuxD.

**ArcoLinuxB** is an ISO that you burn on an usb and a complete Operating System is extracted on your computer.

**ArcoLinuxD** is a base ISO. You get installation scripts and run those one by one. This obviously takes up much more time.

That is why we prefer to work with the ArcoLinuxB ISO.

**ArcoLinuxB is easier and faster to install.**

**For the tutorials it does NOT matter if you are on ArcoLinuxD or ArcoLinuxB. It is the same desktop.**
