# Installation of Mate on Arch Linux Phase 4

install desktop environment MATE ![](https://www.arcolinuxd.com/wp-content/uploads/2018/04/archlinux-mate-vanilla.jpg) Installing mate requires only one line. Theming and tweaking your Arch Linux system will require more time. All the information about this desktop is here: <https://wiki.archlinux.org/index.php/mate>
    
    
    sudo pacman -S mate mate-extra

Video: <https://youtu.be/nIqW-n6vxY8>

Let us go over every settings of Arch Linux Mate and see what we still need to fix. We have a vanilla Arch installation and we keep installing elements like : 

  * terminal
  * theme
  * icon theme
  * text editor
  * conky and Aurora conky (OLD and NOT maintained)
  * Arch Linux wallpapers
  * plank and ArcoLinux themes
  * mate-tweak
  * ...

This time we do **NOT** run the scripts from <https://github.com/arcolinuxd/arco-mate>. You will have to figure things out yourself with the wiki from Arch Linux, Youtube and ofcourse ArcoLinux and ArcoLinuxD. ![](https://www.arcolinuxd.com/wp-content/uploads/2018/04/archlinux-mate-themed.jpg)
