# All in one Arch Linux installation UEFI with Budgie

![](https://www.arcolinuxd.com/wp-content/uploads/2018/12/archlinux-budgie.jpg)

Another example how to install Arch Linux from A till Z.

This time I use a different formatting system: [**btrfs**](<https://wiki.archlinux.org/index.php/Btrfs>).

fat32, swap and root as partitons

Later we install [**budgie**](<https://wiki.archlinux.org/index.php/Budgie>) and [**gnome**](<https://wiki.archlinux.org/index.php/GNOME>) (for more support).

> The Arch Linux ISO of 07/2020 is a special edition.

Many things have been changed and deleted. The archISO package (software to create the Arch Linux ISO) got also a major update.

We have made the comparISOn between June and July 2020 during our Carli 7 project.

You can learn more on these matters on [arcolinux.com](<https://arcolinuxISO.com/category/carli7/>).
