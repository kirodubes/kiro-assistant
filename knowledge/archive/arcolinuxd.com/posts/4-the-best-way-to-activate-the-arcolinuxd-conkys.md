# 4 The best way to activate the ArcoLinuxD conky's

![](https://www.arcolinuxd.com/wp-content/uploads/2018/01/archmerged-mate-aureola-nemesis.jpg) It all boils down to ONE SETTING. And loosing some comfort and gaining a conky. **We need to set "Show Desktop Icons" OFF.** That is the best and easiest solution. Then you start conkyzen and choose your conky. ![](https://www.arcolinuxd.com/wp-content/uploads/2018/01/desktop-icons-mate.jpg) Use CTRL + ALT + TAB to shut down the xserver and log back in. The stability of your system (when using conky's) also depends on the fact if you are on **marco** or **compiz**. Not all conky's will react the same on this desktop as we are used to on Xfce or Openbox depending if you choose marco or compiz.
