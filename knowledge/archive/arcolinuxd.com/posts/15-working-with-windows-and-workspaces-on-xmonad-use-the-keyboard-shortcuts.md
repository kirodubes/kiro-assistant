# 15 working with windows and workspaces on xmonad - use the keyboard shortcuts

![](https://www.arcolinuxd.com/wp-content/uploads/2019/01/arcolinuxb-xmonad-moving-windows.jpg)

We go over the many (but not all) possibilities in Xmonad.

**Super + space** will change the layout in Xmonad

**Alt + Tab** and **Super + Tab** will change the workspace

**Super + Shift + space** will set the layout back to default after a super + space

**Ctrl + Shift + t** will put a floating window back in tiling
