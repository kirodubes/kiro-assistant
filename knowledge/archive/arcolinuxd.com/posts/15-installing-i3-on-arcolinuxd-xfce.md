# 15 - Installing i3 on ArcoLinuxD Xfce

Coming from ArcoLinux we have Xfce, Openbox and i3 on our system.

What if you wanted to have a distro with Xfce and i3 and NOT openbox.

We will install i3 on ArcoLinuxD Xfce using the scripts already created to install [ArcoLinuxD-i3](<https://github.com/arcolinuxd/arco-i3>).

Then you will end up with two desktop environments.
