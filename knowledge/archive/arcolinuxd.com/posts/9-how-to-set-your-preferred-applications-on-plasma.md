# 9 How to set your preferred applications on Plasma

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/default-applications.jpeg)

Every desktop has its own way of ensuring that you can set your preferred application when you click on an url, open a terminal or a file manager.

On Plasma you need to type "default" or "applications" in the menu and you will get a settings windows where you can set 

  * email client
  * file manager
  * terminal
  * browser



In the picture beneath I show you where the default applications settings reside inside the **system settings**.

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/preferred-application-vivaldi.jpeg)
