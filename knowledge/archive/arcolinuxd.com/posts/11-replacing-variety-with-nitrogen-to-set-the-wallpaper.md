# 11 Replacing variety with nitrogen to set the wallpaper

![](https://www.arcolinuxd.com/wp-content/uploads/2018/03/arcolinux-awesome-nitrogen.jpg)

Variety changes the wallpapers but also provides new wallpapers.

Nitrogen changes the wallpapers but does NOT provide new wallpapers.

Your updates for Awesome will come in at **/etc/skel**. Then we need to compare what we have and what comes in.

Use **meld**.

We added the line **run nitrogen --restore**.

We want to give you the possibility to use only nitrogen as wallpaper setter.
