# 14 - Changing the look of the terminal termite on any desktop

When we install the package **arcolinux-termite-themes-git** we will get 79 termite themes. The themes will be installed in **/etc/xdg/termite**.

What we need to do manually now, is to **copy/paste** those **themes** over to our personal **~/.config/termite** folder to work **efficiently**.

Then we choose a theme and copy/paste the color configuration over into the current termite config and save it. Then you test it starting termite.

In the tutorial we will explain you where these 79 themes are hidden, copy/paste them and show you how to switch from theme to theme.
