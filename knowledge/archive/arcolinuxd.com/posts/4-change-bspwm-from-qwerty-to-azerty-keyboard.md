# 4 change bspwm from qwerty to azerty keyboard

![](https://www.arcolinuxd.com/wp-content/uploads/2018/05/arcolinux-bspwm-azerty.jpg)

This article is only for people that have a different keyboard. Standard setting is always **qwerty**.

> I want to have an azerty keyboard.

Living in Belgium I need to change not only autostart.sh but also in **sxhkdrc**. Calamares normally takes care that you have azerty and you do not need to set in autostart.sh.

We search for **azerty** in sxhkdrc and **search** for the word **azerty**.

Scripts will differ with the video in the future.
