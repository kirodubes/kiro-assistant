# 2 How to install compiz on ArcoLinuxD Mate

![](https://www.arcolinuxd.com/wp-content/uploads/2018/01/archmerge-mate-compiz-bis.jpg) WARNING : THINGS MAY BREAK when installing compiz 

If you already followed [this article](<../1-installation-of-arcolinuxd-mate/>) you will have compiz already installed. I wanted to have a separate article about how to install compiz on your ArcoLinux Mate system. It is a question many of the linux users struggle with.

You can either use the **scripts** and **profile** we provide on <https://github.com/arcolinuxd/arco-mate> or just install it yourself with
    
    
    yaourt -S compiz-manjaro

**BEWARE** : getting the settings just right will probably get your system **crashed**. A hard reboot of your system is often the solution.
