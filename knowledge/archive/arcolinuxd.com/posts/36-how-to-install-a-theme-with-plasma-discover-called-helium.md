# 36 How to install a theme with plasma-discover called Helium

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/discover-helium.jpeg)

There are lots of themes but are any of them 'good' and will they work on Plasma is a common question.

I suggest you google around and see what themes pop-up and see if you like them at all.

Then you can decide if you install them. There are different ways to install them

## Installation via plasma-discover

We decided to give **Helium** a try and will install it together.

Press the button **INSTALL**.

This application will do exactly what we did manually in previous article (see link below). It will download, extract and put the contents where it needs to go in

~/.local/share/plasma/desktoptheme

and then it will come available in the system settings of Plasma.

[ In this article we have installed Helium manually from kde.store.org.](<https://www.arcolinuxd.com/35-how-to-install-a-theme-from-store-kdg-org-called-helium/>)

Video: <https://youtu.be/Nsu-iflH1gs>
