# 5 Starting Neofetch automatically in the terminal - any desktop

![](https://www.arcolinuxd.com/wp-content/uploads/2018/03/arcolinux-awesome-neofetch-to-terminal.jpg)We add neofetch to the terminal. We want to see to see the technical information Neofetch provides and the logo of Arcolinux. Add neofetch to **.bashrc** (or .zshrc). 

W3m and imagemagick need to be installed to have the logo of ArcoLinux in Neofetch
