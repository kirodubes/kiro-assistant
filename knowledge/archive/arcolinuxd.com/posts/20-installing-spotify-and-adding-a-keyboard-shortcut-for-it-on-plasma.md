# 20 installing spotify and adding a keyboard shortcut for it on plasma

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/spotify.jpeg)

Check the version of spotify on  
AUR and 3Party repo  
Install the most recent version

Since we use **spotify** constantly on all our installations of all the desktops we have added spotify to the 3party repo. This version will not be auto updated. It is a manual job. One we tend to forget. It is not that important anyway. Spotify will keep working whether it is version x or y.

The **advantage** of having a 3party repo is the **installation** **time**. It will be faster as a build from AUR.

**You decide to install it from AUR or from 3party repo.**  
**Check the version number.**
    
    
    sudo pacman -S spotify
    yay -S spotify
