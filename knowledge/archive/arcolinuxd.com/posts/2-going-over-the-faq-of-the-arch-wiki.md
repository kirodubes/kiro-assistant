# 2 Going over the FAQ of the Arch Wiki

![](https://www.arcolinuxd.com/wp-content/uploads/2018/04/archwiki-faq.jpg)

[I remember the following elements when reading this page.](<https://wiki.archlinux.org/index.php/Frequently_asked_questions>)

  * Arch Linux is a Do-It-Yourself distro
  * a graphical installer is probably not going to be created
  * you need 64 bits hardware
  * it is a rolling release
  * it follows the standard hierarchy standard for the file system
  * is Arch stable? The everlasting question ...
