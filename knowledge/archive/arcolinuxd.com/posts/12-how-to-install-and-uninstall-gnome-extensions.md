# 12 How to install and uninstall gnome extensions

![](https://www.arcolinuxd.com/wp-content/uploads/2018/02/archmerged-gnome-extensions-install-uninstall.jpg)

We use the website to install and uninstall extensions.

<https://extensions.gnome.org>

You can see on this website what extensions are installed on your system. It will only work if you installed the extension in your browser for the gnome extensions. (purple lines at top of page)

**Your extensions are installed in ~/local/share/gnome-shell.**

We install the extra extension for gnome extensions in **Chromium**. It works in **firefox** and **vivaldi** too.

We install a random extension and see how extensions work and where to find it.

How to uninstall extensions and where to change settings will be shown in the video.
