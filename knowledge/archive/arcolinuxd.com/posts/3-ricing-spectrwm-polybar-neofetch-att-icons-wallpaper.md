# 3 Ricing spectrwm - polybar - neofetch - ATT - icons - wallpaper

![](https://www.arcolinuxd.com/wp-content/uploads/2020/12/spectrwm-ricing.jpg)

We change the look of **neofetch** via ArcoLinux Tweak Tool. With the same tool we choose the color scheme of **termite**.

The keyboard shortcuts are there but not the packages. So we install more **applications**. The idea is to start you off with a minimal amount of packages.

We install one of the **Sardi** icons. Thunar will change its folders.

We investigate **Polybar** and show you [where](<https://arcolinux.com/category/arcolinux/design/polybar/>) we can find information about polybar.

We will make the polybar **transparent**. Here it is not interesting as there is a bar hidden behind it.
