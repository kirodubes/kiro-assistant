# 15 Adding keyboard shortcuts to plasma for any application on plasma - variety as example

![](https://www.arcolinuxd.com/wp-content/uploads/2018/11/keyboard-shortcuts-plasma.jpeg)

We have **not** installed **any** **ArcoLinux** keyboard shortcuts on Plasma.

**Plasma** has already **quite a** **variety** of shortcuts and it would mean that we would be overwriting the standard Plasma ones with the ArcoLinux ones.

> We leave it up to the users to decide what shortcuts they need. 

**Make them once.**

**Export** them for later use.

**Import** them again after a clean install.

You can make a keyboard shortcut for any application.

We use **variety** as an example.

**variety --help** gives you all the possible commands for this particular application.

Variety -n shows the next wallpaper

Variety -p shows the previous wallpaper

Variety -f moves the wallpaper to your favorites folder in ~/.config/variety

Variety -t trashes the wallpaper

Variety --toggle-pause will keep the current wallpaper until you press again any of the above.

[In this article](<https://www.arcolinuxd.com/16-exporting-and-importing-your-personal-keyboard-shortcuts-on-plasma-great-after-a-clean-install/>) we will export and import these keyboard shortcuts to be used on a later date.
