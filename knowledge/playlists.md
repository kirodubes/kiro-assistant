# Kiro Playlists (topic-organized video collections)

Erik Dubois groups his videos into topic playlists. Use this file to hand
a user a whole playlist when they want to go deep on one subject.

**Serve order — always try videos before playlists:**

1. **Tier 1** — the KIRO series in [videos.md](./videos.md).
2. **Tier 2** — the older kiro videos in [videos.md](./videos.md).
3. **Tier 3** — the topic playlists below. These are largely
   ArcoLinux-era recordings, but Kiro is Arch-based and runs the same
   apps, so they still apply directly. Reach for them when tiers 1–2 have
   no close match, or when the user wants a deep dive on one topic.

Tool videos (ATT especially) are often filmed on other Arch-based distros
— EndeavourOS, Garuda and friends. The tool is the same, so do not dismiss
a video because of the distro in its title.

Channel: https://www.youtube.com/@ErikDubois
